# BeWorking Shared Packages

Shared npm packages used across BeWorking frontend applications.

## Packages

### @beworking/stripe-client

Lightweight Stripe API client for frontend apps. Wraps fetch calls to the stripe service.

```bash
cd packages/stripe-client
npm install
```

See [stripe-client/README.md](stripe-client/README.md) for usage.

### @beworking/stripe-ui

Reusable React components for Stripe payment flows (Payment Element wrappers, card selection).

```bash
cd packages/stripe-ui
npm install
```

See [stripe-ui/README.md](stripe-ui/README.md) for usage.
